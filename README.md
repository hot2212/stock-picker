# 📈 短线选股系统

基于**热门题材 + 资金流向 + 技术形态**的 A 股短线选股辅助系统。

## 核心功能

| 功能 | 时间 | 说明 |
|------|------|------|
| 🌅 **开盘选股** | 9:27 | 集合竞价后，综合竞价数据+板块热点+均线趋势，选出前5支 |
| 🌆 **尾盘选股** | 14:30 | 埋伏次日有潜力的票，关注资金流向+尾盘抢筹信号 |
| ⭐ **自选股分析** | 随时 | 手动添加股票，系统给出评分、买卖区间、目标价、止损价 |
| 🔄 **自动优化** | 每周六/日 | 分析历史胜率，自动调整选股参数 |
| 📊 **准确率看板** | — | 胜率统计、收益曲线、每日走势 |

## 快速开始

### 1. 环境要求

- Python **3.9+**
- Node.js **18+**

### 2. 安装

```bash
# 后端
cd backend
pip install -r ../requirements.txt

# 前端
cd ../frontend
npm install
```

### 3. 启动

**方式一：分开启动（推荐开发用）**

```bash
# 终端1：启动后端
cd backend
python main.py

# 终端2：启动前端
cd frontend
npm run dev
```

前端访问：http://localhost:3000

**方式二：一键启动（推荐日常用）**

```bash
# Windows：双击 start.bat
# 或命令行：
start python backend/main.py
start npm --prefix frontend run dev
```

### 4. 每天你需要做的事

```
1. 开机
2. 运行 start.bat（或打开两个终端启动）
3. 浏览器打开 http://localhost:3000
4. 看结果
```

## 系统时间线

| 时间 | 系统行为 |
|------|---------|
| 9:25 | 采集竞价数据 |
| **9:27** | 🌅 **开盘选股，推送结果** |
| 9:30~11:30 | 盘中监控，每30秒刷新实时行情 |
| 13:00~14:25 | 午后监控 |
| **14:30** | 🌆 **尾盘选股，推送结果** |
| 15:00 | 记录收盘价，更新胜率 |
| 周末 10:00 | 自动优化参数，生成周报 |

## 页面说明

- **📊 今日看板** — 早盘5支 + 昨日尾盘5支 + 今日尾盘5支(14:30后) + 板块排行 + 题材热度
- **⭐ 自选股分析** — 手动添加自选，系统分析买卖区间/目标价/止损价
- **📅 历史记录** — 近30天选股记录，展开可看详情
- **📈 准确率看板** — 胜率曲线、收益统计、每周优化报告

## 技术栈

| 层 | 技术 |
|----|------|
| 后端 | Python FastAPI + APScheduler |
| 前端 | Vue3 + Vite |
| 数据库 | SQLite |
| 数据源 | 腾讯财经 + 同花顺 + 东方财富 + 百度股市通 |

## 数据保留

- 早盘/尾盘选股：保留 **30天**，每月自动清理
- 自选股分析：**手动删除**

## 参数调优

所有选股阈值集中在 `backend/config.py`，自我优化模块每周六/日自动分析数据并调整。
也可手动修改该文件中的参数值。
